# CraftUI API

Express.js API server built with TypeScript.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Production
```bash
npm start
```

### Linting
```bash
npm run lint
npm run lint:fix
```

## Project Structure
```
src/
├── index.ts       # Entry point
└── ...            # Add more modules as needed
dist/              # Compiled JavaScript (generated)
```

## Technologies
- Express.js
- TypeScript
